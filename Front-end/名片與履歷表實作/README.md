解題想法與概念：
1. 由上到下去畫畫圖，用紙筆畫出你有幾個畫面，想怎樣呈現
2. 定義好你有哪些大區塊，想好各區塊要用什麼 class 去定義 CSS
3. 做大區塊中的小區塊定義，一樣想用什麼 class定義CSS
4. 由上到下慢慢修飾，會慢慢有輪廓，不要細修，先雕輪廓
5. 細修與調整細節
